# 链表A+B
 - 题目描述
 - 编写代码，以给定值x为基准将链表分割成两部分，所有小于x的结点排在大于或等于x的结点之前
 - 给定一个链表的头指针 ListNode* pHead，请返回重新排列后的链表的头指针。注意：分割以后保持原来的数据顺序不变。

# 算法
 - 建两个链表表头一个大一个小。然后比较分别放入大小两个链表中。然后连接大小两个链表。最后返回小链表表头的后一个节点
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(N)。
 - 这个算法最短C++ 23行，python 19行
