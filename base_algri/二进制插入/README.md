# 二进制插入

 - 题目描述
 - 有两个32位整数n和m，请编写算法将m的二进制数位插入到n的二进制的第j到第i位,其中二进制的位数从低位数到高位且以0开始。

 - 给定两个数int n和int m，同时给定int j和int i，意义如题所述，请返回操作后的数，保证n的第j到第i位均为零，且m的二进制位数小于等于i-j+1。

 - 测试样例：
 - 1024，19，2，6
 - 返回：1100


# 算法
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(1)。
 - 这个算法最短C++ 3行，python 2行 java 3行
