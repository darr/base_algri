# 最接近的数
 - 题目描述
 - 有一个正整数，请找出其二进制表示中1的个数相同、且大小最接近的那两个数。(一个略大，一个略小)

 - 给定正整数int x，请返回一个vector，代表所求的两个数（小的在前）。保证答案存在。

 - 测试样例：
 - 2
 - 返回：[1,4]


# 算法
 - 只观的解法 然后加1寻找大 减1寻找小 结构清晰 代码略长 这个题虽然是关于位运算的 但是感觉直观的解法更简单，并不费神。重要的是时间的复杂度不高，答案应该能快速找到
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(N)。
 - 这个算法最短C++ 16行，python 16行
