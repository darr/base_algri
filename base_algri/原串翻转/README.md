# 原串翻转
 - 题目描述
 - 请实现一个算法，在不使用额外数据结构和储存空间的情况下，翻转一个给定的字符串(可以使用单个过程变量)。

 - 给定一个string iniString，请返回一个string，为翻转后的字符串。保证字符串的长度小于等于5000。

 - 测试样例：
 - "This is nowcoder"
 - 返回："redocwon si sihT"



# 算法
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(N)。
 - 这个算法最短C++ 8行，python 6行
