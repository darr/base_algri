# 平分的直线
 - 题目描述
 - 在二维平面上，有两个正方形，请找出一条直线，能够将这两个正方形对半分。假定正方形的上下两条边与x轴平行。

 - 给定两个vecotrA和B，分别为两个正方形的四个顶点。请返回一个vector，代表所求的平分直线的斜率和截距，保证斜率存在。

 - 测试样例：
 - [(0,0),(0,1),(1,1),(1,0)],[(1,0),(1,1),(2,0),(2,1)]
 - 返回：[0.0，0.5]




# 算法
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(N)。
