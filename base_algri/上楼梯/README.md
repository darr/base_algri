# 上楼梯
 - 题目描述
 - 有个小孩正在上楼梯，楼梯有n阶台阶，小孩一次可以上1阶、2阶、3阶。请实现一个方法，计算小孩有多少种上楼的方式。为了防止溢出，请将结果Mod 1000000007

 - 给定一个正整数int n，请返回一个数，代表上楼的方式数。保证n小于等于100000。

 - 测试样例：
 - 1
 - 返回：1



# 算法
 - 没有借助额外的数据结构或者内存。
 - 时间复杂度O(N)。
 - 这个算法最短C++ 9行，python 9行
