# 寻找下一个结点
 - 题目描述
 - 请设计一个算法，寻找二叉树中指定结点的下一个结点（即中序遍历的后继）。
 - 给定树的根结点指针TreeNode* root和结点的值int p，请返回值为p的结点的后继结点的值。保证结点的值大于等于零小于等于100000且没有重复值，若不存在后继返回-1。


# 算法
 - 借助一个额外的结点 利用递归，先判断左子树，然后自己，然后右子树。
 - 时间复杂度O(N)。
 - 这个算法最短C++ 10行，python 10行 java 10行
